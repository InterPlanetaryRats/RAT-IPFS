<h1 align="center">
  RAT-IPFS
</h1>
RAT-IPFS is a p2p remote access tool built on <a href="https://github.com/Stack-Box/stackbox/actions?query=workflow%3AVue" alt="Vue">ipfs</a> which can be used to access peer nodes anywhere over the internet.
